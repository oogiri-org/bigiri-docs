# bǃgǃrǃ ユーザーマニュアル

## 1. はじめに
本マニュアルは、大喜利プラットフォーム「bǃgǃrǃ (oogiri.org)」の基本的な使い方から各種機能について解説した公式ユーザーマニュアルです。

## 2. アカウントとサインイン
- **サインイン方法**:
  通常のサインインに加えて、**X (旧Twitter)** および **Bluesky** アカウントを利用したOAuthサインインに対応しています。ソーシャルサインインを行うと、入室者一覧からプロフィールへのリンクが表示されます。
- **名前の設定**:
  初期設定の名前は「お客さん」です。トップ画面や部屋画面のメニューから自身の名前を変更できます。
  - **部屋内での一時的な名前変更**: 部屋の中で名前を変更した場合、その部屋でのみ名前が変更され、ソーシャルリンクは表示されなくなります。
  - **名前のハッシュ化**: 名前の後ろに `#` と任意の文字列（パスワード）を付けると、暗号学的ハッシュ値（8文字）に変換して表示されます。（例: `名前#abc` → `名前#DnbJh2JQ`）
- **サインオーバー（引き継ぎ）機能**:
  現在のサインイン状態を別の端末やブラウザーに引き継ぐことができます。メニューから「サインオーバー」を選択し、表示されるURLやQRコードを1時間以内に引き継ぎ先で開くことで完了します。

## 3. トップ画面と部屋の作成
- **部屋一覧**:
  トップ画面には現在存在する部屋が一覧表示されます。「作成順」や「更新順」で並び替えが可能です。Twitterハッシュタグ検索へのリンクや、設定されているアイコン画像も確認できます。
- **部屋の作成**:
  自分で新しい部屋（大喜利ルーム）を作成することができます。
  - **設定履歴機能**: 過去に同じブラウザーで作成した部屋の設定履歴を読み込むことができます。
  - **Discord通知**: 新しい部屋が作成されたことを、公式Discordサーバーや指定したWebhook URLへ通知する設定が可能です。
- **カラースキームの設定**:
  画面の配色を「オート・ライト・ダーク」から選択できます。部屋作成時にデフォルトを指定できるほか、参加者各自がメニューバーから一時的に切り替えることも可能です。

## 4. 部屋の基本機能とグループ
- **入室者一覧**:
  部屋にいるメンバーを確認できます。「先入順」「後入順」、獲得したユニコーンの「多🦄順」「少🦄順」など、多彩なソートが可能です。
  - 🔰 **初心者マーク**: 登録から24時間以内のユーザーに表示されます。
  - **受刑者（キックアウト）**: 部屋作成者は、不適切な入室者を「受刑者」に指定して強制退室させることができます。
- **グループ機能**:
  部屋作成者は部屋内に「グループ」を作成し、参加者をまとめることができます。
  - グループを使用することで、投票時に「自分のグループ内のみの回答を表示する」「次のグループの回答に投票する」といった**グループ分離**の設定が可能となり、大人数の部屋でもスムーズに進行できます。
- **便利なキーボードショートカット**:
  - `Ctrl + Enter` または `Meta (Windows/Command) + Enter` : テキスト入力の送信
  - `Esc` : ダイアログウィンドウを閉じる

## 5. ゲームの進行
### 5.1 出題フェーズ
- **お題の投稿**:
  最大1500文字までのお題を投稿できます。10行を超える部分は「＋」ボタンで折りたたまれます。
  出題には画像を添付することも可能で、他者の著作物等を用いる場合は「出典」欄に記載できます。
- **出題優先時間**:
  部屋の設定により「部屋作成者のみ」や「前回の1位のみ」が優先して出題できる時間が設けられることがあります。優先される出題者の「出題する」ボタンは明滅してお知らせします。

### 5.2 回答フェーズ
- **回答の投稿**:
  最大1500文字まで入力可能です。テキストだけでなく、設定で許可されていれば**手描きの線画**を用いた回答も行えます。
- **回答シャッフル時間**:
  部屋設定で「回答シャッフル時間」が設けられている場合、その時間内に投稿された回答は、投票フェーズにて全員にランダムな順番で表示されます。
- **自回答フィルター**:
  自分の回答のみを絞り込んで表示する機能です。

### 5.3 投票フェーズ
- **投票の操作**:
  回答に対して票を投じます。
  - **クイック0票投票**: 「投票する」ボタン横の「0」ボタンで、簡単に0票を投票できます。
  - **キーボード操作**: 投票ボタンにフォーカスがある状態で、`Space` キーで増票、`Delete` キーで減票、`Enter` キーで送信ができます。
- **投票制限と超過可**:
  1題ごと、または1回答ごとの投票数制限が設定されています。設定が「超過可」となっている場合、制限の基準を超えて投票を行うことも可能です。

### 5.4 結果・履歴フェーズ
- **順位と評価の表示**:
  集計結果に基づいて順位が発表されます。同票の場合は「投票に多く参加した回答者」が優先され、さらに同数の場合は「回答の早さ」で決定します。
  また、補正得票数のZスコア（偏差値基準）に基づき、以下の特別な装飾が付与されます。
  - **Zスコア 1以上** : 順位に下線が表示
  - **Zスコア 2以上** : 🦄 ユニコーンが表示（さらに1上がるごとに1つ追加）
  - **Zスコア -1以下** : 順位に上線が表示
  - **Zスコア -2以下** : 🍦 アイスクリームが表示（さらに1下がるごとに1つ追加）
- **コメントとリアクション**:
  出題や回答に対してコメントを付けることができます。スマイルアイコンのボタンからは「クイックコメント」として手軽に絵文字リアクションを送信できます。
- **過去の結果とログ**:
  過去の出題や結果（最大34.5日分）を振り返ることができます。また、詳細な記録をCSV形式でダウンロード可能です。CSVには名前やグループ名、ディスクリミネーター（個別識別子）が記録されます。
