## 1. bǃgǃrǃについて

bǃgǃrǃ（ビギリ）は、ブラウザーで大喜利を遊ぶためのWebアプリケーションです。参加者が同じ「部屋」に入り、次の流れを繰り返します。

1. 出題者がお題を投稿します。
2. 回答者がお題への回答を投稿します。
3. 投票者が面白い回答へ投票します。
4. 結果画面で順位を確認し、コメントやリアクションを投稿します。
5. 次のお題を出して新しい回を始めます。

部屋ごとに、参加できる役割、回答や投票の制限、匿名表示、制限時間、グループなどを設定できます。スマートフォンなど幅の狭い画面にも対応しています。

### 利用前に知っておくこと

- 操作にはサインインが必要です。
- 部屋のルールは部屋ごとに異なります。操作できないときは、現在のフェーズ、制限時間、参加者制限を確認してください。
- 部屋は最終更新から11.5日で終了します。
- 過去の結果とCSVログは、34.5日以内の出題が対象です。

## 2. 基本的な遊び方

### 2.1 参加者として遊ぶ

1. トップ画面でサインインします。
2. 部屋一覧から参加する部屋を選びます。
3. 必要に応じてグループを選びます。
4. お題が表示されたら回答します。
5. 投票フェーズで回答に投票します。
6. 結果フェーズで順位、コメント、リアクションを確認します。

部屋作成者が出題者・回答者・投票者を限定している場合、該当する役割に指定されていない利用者はその操作を行えません。

### 2.2 部屋作成者として進行する

部屋作成者は通常の参加に加えて、次の操作を行えます。

- 部屋作成時のルール設定
- 回答・投票・出題優先時間の締め切り
- 部屋コメントの投稿と、最新の部屋コメントの削除
- 入室者ごとの出題者・回答者・投票者の指定
- グループの作成と削除
- 受刑者への追加と解除

## 3. サインインと利用者情報

### 3.1 名前でサインインする

サインイン画面で名前を入力します。名前の初期値は「お客さん」です。サインイン時にはreCAPTCHAによる確認が行われます。

### 3.2 名前にハッシュ値を付ける

名前の末尾に `#文字列` を付けると、`#` 以降が8文字のハッシュ値に変換されます。同じ文字列からは同じ値が生成されるため、同名利用者の区別や継続的な本人確認の目印として利用できます。

```text
びぎりちゃん          → びぎりちゃん
びぎりちゃん#abc      → びぎりちゃん#DnbJh2JQ
風祭ビギリマナ春子#abc → 風祭ビギリマナ春子#C2GkNEL6
```

ハッシュ値は秘密情報そのものを表示しない仕組みですが、推測されやすい文字列の利用は避けてください。

### 3.3 ソーシャルサインイン

#### Twitter（X）

Twitterアカウントでサインインできます。

- 入室者一覧にプロフィール画像やアカウントへのリンクを表示できます。
- プロフィール画像を読み込めない場合は、デフォルト画像が表示されます。
- 通常サインインでは未出題の部屋を1つまで作成できますが、Twitterサインイン時は未出題の部屋を複数作成できます。
- Twitterサインインの画面遷移には暫定的な措置が適用されており、予告なく変更される場合があります。

#### Bluesky

BlueskyアカウントによるOAuthサインインを利用できます。この機能は試験稼動です。トップ画面や部屋画面では、Twitterの投稿・検索リンクと同等のBlueskyリンクも表示されます。

### 3.4 名前を変更する

トップ画面または部屋画面のメニューに表示された自分の名前を押すと、サインアウトせずに名前を変更できます。

- 部屋内で変更した名前は、その部屋でのみ使われます。
- 部屋画面で自分の名前を押すと、入室者一覧に表示するソーシャルリンクのURLと表示の有無も設定できます。
- 部屋内で名前を変更している場合や、名前にハッシュ値を付けている場合も、ソーシャルリンクは現在の設定に従います。

### 3.5 別のブラウザーへ引き継ぐ（サインオーバー）

1. トップ画面または部屋画面のメニューから「サインオーバー」を選びます。
2. 表示されたURLをコピーするか、QRコードを読み取ります。
3. 1時間以内に引き継ぎ先のブラウザーでURLを開きます。

URLを知っている人がサインイン状態を引き継げるため、第三者へ共有しないでください。

## 4. トップ画面と部屋への入室

トップ画面には公開中の部屋一覧、更新情報、各種メニューが表示されます。

### 4.1 部屋を探す

部屋一覧では、部屋名、説明、アイコン画像、出題日時、入室者名などを確認できます。自分の名前は一覧上で判別できるように表示されます。

表示順はドロップダウンリストから選べます。

| 表示順 | 内容 |
|---|---|
| 作成順 | 作成日時が新しい部屋から表示 |
| 更新順 | 更新日時が新しい部屋から表示 |

### 4.2 自分が入室している部屋だけを見る

部屋一覧の先頭にある手のアイコン（自入室フィルター）を押してオンにすると、自分が入室している部屋だけが表示されます。もう一度押してオフにすると、全ての部屋の表示に戻ります。

### 4.3 部屋へ入る

参加したい部屋を選んで部屋画面へ移動します。部屋内では、現在のフェーズに応じて出題、回答、投票、結果の操作が表示されます。

### 4.4 更新情報などを見る

- 「更新情報を見る」ボタンを押すと、更新履歴がダイアログで表示されます。
- ダイアログはエスケープキーでも閉じられます。
- トップ画面にはGitHub Discussionsへのリンクがあります。
- 新着の部屋はAtomフィードでも購読できます。フィードには部屋作成者名が含まれます。

## 5. 部屋を作成する

トップ画面から部屋作成画面を開き、部屋名、説明、アイコン画像、ゲームのルールを設定します。

### 5.1 基本情報

- **部屋名**: 一覧や部屋画面に表示される名前です。
- **説明**: 部屋の目的やルールを記入します。説明内のURLは自動的にリンクになります。
- **アイコン画像**: 部屋の右下などに表示する画像です。画像には容量制限があります。

### 5.2 作成履歴を再利用する

部屋名の横にある履歴ボタンから、同じブラウザーで以前作成した部屋の設定を利用できます。

1. 履歴ボタンを押します。
2. 作成日時と部屋名を確認します。
3. 「読み込む」で設定をフォームへ反映するか、「削除する」で履歴を消します。

同じ部屋名で複数回作成した場合、履歴には最新の設定だけが残ります。直近の設定値は、アイコン画像を含めて次回作成時の初期値になります。

### 5.3 未出題の部屋数

- 通常サインインでは、まだ出題していない部屋は1つまで作成できます。
- Twitterサインインでは、未出題の部屋を複数作成できます。

## 6. 部屋の設定

設定項目はゲーム中の表示と権限に直接影響します。参加者へ事前にルールを案内してから開始してください。

### 6.1 参加者の制限

出題者・回答者・投票者の各制限には、制限無し、制限有り、ソーシャルサインイン利用者に限定する設定があります。これらは初期設定です。

部屋作成後に入室者一覧で特定の利用者を出題者・回答者・投票者へ追加または削除すると、その役割については入室者一覧の個別指定が優先されます。

### 6.2 出題の設定

| 項目 | 内容 |
|---|---|
| 出題画像制限 | お題への画像添付を禁止できます。 |
| 出題優先時間 | 結果発表後の一定時間、前回の投票者中1位または部屋作成者だけが出題できるようにします。 |

出題優先時間中は、優先される人の「出題する」ボタンが明滅します。部屋作成者は優先時間を途中で締め切れます。

### 6.3 回答の設定

| 項目 | 内容 |
|---|---|
| 回答数制限 | 1つのお題に投稿できる回答数を制限します。 |
| 回答時間制限 | 回答フェーズの制限時間を設定します。 |
| 回答者名表示 | 回答者名の表示方法を設定します。自分の回答者名は本人には常に表示されます。 |
| 回答表示 | 投稿後すぐに表示する「即時」と、締め切り後に表示する「一斉」を選びます。 |
| 回答シャッフル時間 | 指定時間内の回答を、投票時に入室者ごとに異なるランダム順で表示します。秒数を空にすると全回答が対象です。 |
| 回答描画 | 回答への手描き線画の利用を制限します。 |

回答シャッフルは投票時の表示順に作用し、結果フェーズの順位には影響しません。

### 6.4 投票の設定

| 項目 | 内容 |
|---|---|
| 投票時間制限 | 投票フェーズの制限時間を設定します。「回答毎×秒」で回答数に応じた時間を加算できます。 |
| 投票者名表示 | 投票者名の表示方法を設定します。自分の投票者名は本人には常に表示されます。 |
| 投票表示 | 投票をすぐ反映する「即時」と、締め切り後に反映する「一斉」を選びます。 |
| 投票数制限（出題毎） | 1つのお題全体に使える票数を制限します。 |
| 投票数制限（回答毎） | 1つの回答へ入れられる票数を制限します。 |
| 自回答投票 | 自分の回答への投票を許可するか設定します。 |
| 投票グループ | 投票時に表示するグループの範囲を設定します。 |

投票数制限（出題毎）の「超過可」では、制限到達後も投票画面から制限を超えて投票できます。

### 6.5 コメントと表示の設定

| 項目 | 内容 |
|---|---|
| 出題コメント制限 | コメントを制限無し、または部屋作成者だけに制限します。 |
| 回答コメント制限 | コメントを制限無し、または部屋作成者だけに制限します。 |
| カラースキーム | 部屋の初期配色をライトまたはダークに設定します。 |

## 7. 出題する

部屋作成直後、または前の結果が確定した後に「出題する」ボタンからお題を投稿します。

### 7.1 お題を入力する

- 入力できる本文は最大1500文字です。
- 複数行入力欄は、入力した行数に合わせて自動的に伸びます。
- 投稿後、150文字を超える部分や10行を超えた位置には残りを表示するボタンが付きます。押すと全文を確認できます。
- 出典欄を利用できます。著作物をお題に使う場合は適切な出典を記載してください。

### 7.2 画像を付ける

部屋の設定で禁止されていなければ、お題に画像を付けられます。本文が未入力の場合、画像ファイル名が本文の初期値になります。出題画像の背景はライト配色と同じ白で表示されます。

### 7.3 投稿する

画面上の送信操作のほか、入力欄で次のキーを使用できます。

- `Ctrl` + 改行キー
- `Windows` キーまたは `Command` キー + 改行キー

複数の利用者が同時に出題した場合は競合処理が行われます。投稿後は画面上の現在のお題を確認してください。

## 8. 回答する

回答フェーズで回答欄を開き、お題への回答を投稿します。

### 8.1 回答を入力する

- 回答は最大1500文字です。
- 複数行入力欄は、入力した行数に合わせて自動的に伸びます。
- 長い回答は、150文字または10行を超えた位置で折りたたまれます。残りを表示するボタンで全文を開けます。
- `Ctrl` + 改行キー、またはメタキー + 改行キーでも送信できます。
- 回答数はリアルタイムで表示されます。

### 8.2 手描き線画を付ける

部屋設定で許可されている場合、回答に手描き線画を追加できます。タッチ操作にも対応しています。

### 8.3 回答を変更・削除する

- 「回答を変更する」から投稿済み回答を編集できます。変更すると回答時刻も更新されます。
- 回答を削除した直後に再び回答すると、削除した内容が回答欄の初期値になります。

「一斉」表示の部屋では、締め切りまで他の参加者の回答が見えない場合があります。

## 9. 投票する

投票フェーズで各回答を確認し、「投票する」から票数を指定します。部屋設定により、使える総票数、1回答あたりの票数、自回答への投票可否が異なります。

### 9.1 通常の投票

1. 投票したい回答の「投票する」を押します。
2. 投票数ボタンで票数を指定します。
3. 送信して投票を確定します。

投票数ボタンには最初からフォーカスが設定されます。フォーカス中は次のキーを使えます。

| キー | 操作 |
|---|---|
| スペース | 1票増やす |
| デリート | 1票減らす |
| 改行 | 投票を送信する |

### 9.2 0票クイック投票

「投票する」の横にある「👍0」ボタンを押すと、投票画面を開かずに0票を投票できます。回答を確認済みであることを明示したい場合などに利用できます。

### 9.3 投票を変更・削除する

投票後は「投票を変更する」から、既存の投票をいったん削除せずに票数を変更できます。投票一覧から不要な投票を削除することもできます。

### 9.4 自分の投票を確認する

- 回答一覧上部の「投票を見る」は、自分が投票した回答だけを表示します。
- 投票を見る画面には、投票数の合計と出題毎の投票数制限が表示されます。
- 回答ごとの得票数「👍n」を押すと、その回答について自分が行った投票だけを表示できます。
- 投票の表示順は票数の降順で、同数の場合は投票順です。

投票数制限に達した場合は投票一覧を開き、投票の削除や配分変更を行ってください。

## 10. 結果と順位を見る

投票が締め切られると結果フェーズになります。回答、得票数、順位、回答者名、投票者名は部屋の設定に従って表示されます。

### 10.1 順位の決まり方

1. 得票数が多い回答が上位になります。
2. 得票数が同じ場合、回答者自身の投票数が多い回答が上位になります。
3. それも同じ場合、投稿時刻が早い回答が上位になります。

### 10.2 Zスコアと順位の装飾

回答者の投票数で補正した得票数のZスコアに応じて、順位に装飾が表示されます。

| Zスコア | 表示 |
|---|---|
| +1以上 | 順位に下線 |
| +2以上 | ユニコーン🦄。さらに1上がるごとに1つ追加 |
| -1以下 | 順位に上線 |
| -2以下 | アイスクリーム🍦。さらに1下がるごとに1つ追加 |

その部屋で獲得したユニコーンの累計は、部屋画面のメニューバーと入室者一覧に表示されます。

## 11. コメントとリアクション

### 11.1 部屋コメント

部屋作成者は、部屋上部に表示される部屋コメントを投稿できます。削除操作では、全件ではなく最新の1件が削除されます。

### 11.2 出題コメント・回答コメント

結果フェーズでは、お題や回答へコメントできます。投稿できる人は部屋設定に従います。

- 過去の結果画面でもコメントを確認できます。
- 空文字列のコメントも受け付けられます。
- 通常の参加者は投稿済みの出題コメント・回答コメントを削除できません。

### 11.3 クイックコメント（リアクション）

出題コメントまたは回答コメントの横にあるスマイルボタンから絵文字を選ぶと、その絵文字をコメントとして素早く投稿できます。用意される絵文字は8種類で、更新により入れ替わる場合があります。

## 12. グループを使う

グループは、同じ部屋の入室者をチームのようにまとめる機能です。グループは作成した部屋の中だけで有効です。

### 12.1 グループを作成する

部屋作成者は「グループを選ぶ」画面からグループを追加・削除できます。

- グループ名は重複でき、空文字列も使用できます。
- グループに説明を設定できます。
- グループ名の横の履歴ボタンから、同じブラウザーで作成した設定を読み込み・削除できます。
- 同じグループ名の履歴は最新の設定だけが残ります。

### 12.2 グループを選ぶ

次の場合、入室者にはグループ選択画面が表示されます。

- 部屋作成者がグループを追加したときに、まだグループへ入っていない場合
- グループ作成後に部屋へ入室した場合
- 自分が選んでいたグループが削除された場合

自分のグループは「グループを選ぶ」画面からいつでも変更できます。

### 12.3 グループの表示

- メニューバーでは、自分の名前の前にグループ名が表示されます。
- 回答者名や入室者名の前にもグループ名が表示されます。
- 結果と過去の結果では、グループ名を押してそのグループの回答だけを表示できます。
- CSVでは回答者名の前と、9番目のカラムにグループ名が記録されます。

### 12.4 グループ分離

部屋作成者がグループを追加すると、回答・投票フェーズでは、自分と異なるグループの回答が表示されないグループ分離が有効になります。グループが存在しない場合は通常どおり動作します。

### 12.5 投票グループ

| 設定 | 投票フェーズで表示される回答 |
|---|---|
| 無し | 通常の範囲の回答 |
| グループ内に投票 | 自分と同じグループの回答 |
| 有り（次グループに投票） | 自分のグループの次に作成されたグループの回答 |

「次グループに投票」で自分が最後のグループの場合は、最初に作成されたグループが対象です。グループ未選択者には、グループ未選択者の回答だけが表示されます。
