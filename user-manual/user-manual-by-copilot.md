## 1. はじめに

bǃgǃrǃ は、出題・回答・投票を通じて参加者が交流できるWebアプリケーションです。部屋を作成し、参加者が様々なお題に対して回答し、投票を行うことで楽しいひとときを過ごすことができます。

### 1.1 基本コンセプト
- **部屋**: 参加者が集まって活動する場所
- **出題**: お題を出すこと
- **回答**: お題に対する答えを投稿すること  
- **投票**: 他の参加者の回答に投票すること
- **グループ**: 参加者を分けて管理する機能

## 2. 基本的な使い方

### 2.1 初回利用時の流れ
1. サインインを行う
2. 部屋を作成するか、既存の部屋に参加する
3. お題が出題されたら回答する
4. 投票フェーズで他の参加者の回答に投票する
5. 結果を確認する
